### Usage Instructions

*Clone this repository* with git command line or Git for Windows or else you can
also *download the repository as a ZIP file*. Then put the project folder
somewhere on your filesystem of your PC. Launch your Eclipse for Java
installation and import it as a Maven project.

The project is meant to be used in conjunction with the
[osp-011-battleship-hid](https://github.com/unibz-makerspace/osp-011-battleship-hid)
project.
